 GoalsBuild a Visual note taking tool.Each Note should be Movable and Editable.Each Note should support Rich Text Capabilities.User should be able to Add, Access, and Remove Different Pages.
 DescriptionVisually organize your notes in a canvas.Each note can be edited, scaled, and colored.Notes support videos, images, code blocks, and more.You can group your notes into seprate pages.
 DetailsThe project was built using HTML, CSS, and Javascript.SvelteJS was used to model the components.The pan and zoom for the canvas and the note transformations were all written from scratch, using the browser's built in capabilities.
